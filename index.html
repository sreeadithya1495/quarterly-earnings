Capital Asset Pricing Model (CAPM)

We use CAPM to determine the expected return on our investments.

$$ R_a = R_f + \beta_a(R_m - R_f) $$

Where:
$R_a$ = Expected Return
$R_f$ = Risk-Free Rate
$\beta_a$ = Beta of the Asset
$R_m$ = Expected Market Return
